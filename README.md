# Enquete.com

<img src="acessorios/tela_inicial.jpg">

EnqueteCom ou Enquete.com, chame do que preferir é um projeto Web que visa possibilitar a criação de enquetes dinâmicas e checagem de votos.

## Do Propósito

É um projeto simples que visa aperfeiçoar minhas habilidades nas tecnologias utilizadas, tanto FrontEnd quanto o Backend.

- Criar e gerenciar enquetes
- Votar em enquetes
- Visualizar estatisticas de enquetes
- Consulta de enquetes publicas
